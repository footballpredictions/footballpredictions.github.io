# Футбольные прогнозы 2.0

Титульная страница Android-приложения «Футбольные прогнозы 2.0». Служит для ознакомления с функциями приложения и скачивания APK.

## Возможности приложения

- **Прогнозы** — актуальные прогнозы на футбольные матчи (дата, время, лига, команды, рекомендуемый исход). Часть прогнозов бесплатна, полный доступ — по подписке.
- **Статистика** — результаты прогнозов: выигрыши, проигрыши, возвраты, процент побед. Бесплатная и платная вкладки.
- **Топ-10** — списки матчей на сегодня по алгоритму: Тотал больше 2.5, Обе забьют, Победы, Тотал Угловых, Тотал Жёлтых карточек, Тотал Фолов.
- **Банкролл** — учёт событий и контроль дохода/расхода. Статистика ставок, ROI, общая прибыль.

## Версия приложения

Текущая версия определяется автоматически:
1. Из GitHub Releases API
2. Из `version.json` (фолбэк для старых устройств)

При новом релизе обновить `version.json`:
```json
{ "version": "X.Y.Z" }
```
